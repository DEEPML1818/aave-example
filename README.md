## Usage

1. Open `.env` and then add your `PRIVATE_KEY` and `RINKEBY_URL`
2. Deploy contract (fuji or rinkeby)

```
yarn deploy:fuji
yarn deploy:rinkeby
```

3. Open `config.js` file and update `simpleFlash` (Deployed address)
4. Run SimpleFlashLoan

```
yarn fl:fuji
yarn fl:rinkeby
```
